QGIS ve Python (PyQGIS) birleşimiyle geliştirilen GeoRehber eklentileri. Lokasyon haritaları, otomatik layout üretimi ve tekrarlayan CBS iş akışlarını hızlandıran araçlar.

GeoRehber Location Map

İl / ilçe verilerinden otomatik olarak lokasyon haritası, sayfa düzeni,kartografik tasarım yapan, lejant, ölçek çubuğu ve yön oku oluşturan QGIS eklentisi.

QGIS 3 için geliştirilmiştir. Kurulum, kullanım dökümantasyonu ve örnek proje dosyaları bu sayfadan paylaşılacaktır.

Yapım Aşamasında

QGIS Eklenti Kurulum Rehberi

1

Eklentiyi yukarıdaki İndir (.zip) butonu ile indirin.

2

QGIS menüsünden Plugins → Manage and Install Plugins... bölümüne gidin.

3

Sol alttaki Install from ZIP seçeneğine tıklayın.

4

ZIP dosyasını seçin.

5

Install Plugin butonuna basın.

Kurulum tamamlandığında QGIS üst menüsünde eklenti aktif olur.
